simnibs.msh¶

This module handles I/O and manipulation of mesh files

I/O Functions¶

read_msh Reads a gmsh ‘.msh’ file
write_msh Writes a gmsh ‘msh’ file
read_freesurfer_surface Function to read FreeSurfer surface files, based on the read_surf.m script by Bruce Fischl
write_freesurfer_surface Writes a FreeSurfer surface Only the surfaces (triangles) are writen to the FreeSurfer surface file
read_gifti_surface Reads a gifti surface
read_curv Reads a freesurfer .curv file
write_curv Writes a freesurfer .curv file
read_stl
write_geo_spheres Writes a .geo file with spheres in specified positions
write_geo_text Writes a .geo file with text in specified positions

Mesh Classes¶

Msh class to handle the meshes.
Nodes class to handle the node information:
Elements Mesh elements.
NodeData Data (scalar, vector or tensor) defined in mesh nodes.
ElementData Data (scalar, vector or tensor) defined in mesh elements.
