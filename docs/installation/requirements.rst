Requirements
============

Operating Systems
-----------------

Windows
''''''''
SimNIBS has been tested on Windows 7 and 10


Linux
''''''

SimNIBS has been tested on Ubuntu 16.04 and 18.04 as well as CentOS 6 and 7.


MacOS
''''''
SimNIBS has been tested on MacOS 10.14 (Mojave)


Hardware
--------

  * 3 GB disk space
  * 6GB of memory (8GB are recommended)
